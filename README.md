# Positions face à l'IA générative dans les arts

Cartographie des positions et contours d'une troisième voie laissée ouverte.

Ce texte recense les postures existantes face à l'IA générative dans le champ artistique (rejet, adoption sans recul, entre-deux) et tente de dessiner les contours d'une position qui ne s'y réduit pas. Il aborde aussi les questions de droit d'auteur, de consentement et d'impact écologique liées à ces outils.
